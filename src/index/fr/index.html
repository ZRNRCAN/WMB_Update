Navigateur cartographique du SATC

Nous avons le plaisir de vous annoncer le lancement de ce nouveau navigateur cartographique du SATC. Les utilisateurs devraient mettre à jour leurs signets vers cette page. L’ancien navigateur cartographique du SATC restera actif jusqu’en septembre 2022. Si vous avez besoin d’assistance en utilisant ce nouvel outil, veuillez nous contacter à l’adresse suivante : clss-satc@nrcan-rncan.gc.ca

Aide - Cliquez pour afficher ou pour masquer l'aide

Le navigateur cartographique du SATC permet aux utilisateurs de parcourir les données cadastrales du Système d'arpentage des terres du Canada (SATC) pour rechercher et pour visualiser les parcelles d'arpentage et les plans qui sont maintenus par la Direction de l'arpenteur général (DAG). Les utilisateurs peuvent naviguer dans la carte avec une souris ou un clavier et rechercher les données avec les recherches attributs.

Guide de l'utilisateur

Veuillez vous référer au Guide de l’utilisateur du navigateur cartographique du SATC (PDF, 2.47 MO) pour plus d'information sur l'application.

Consulter notre document d'information pour vous aider à choisir le bon outil.

Note aux utilisateurs de clavier

La carte peut être contrôlée à l'aide du clavier. Veuillez utiliser la touche de tabulation pour activer le commandes de la carte. Les touches de navigation sont les suivantes:

  • "+": zoom avant
  • "-": zoom arrière
  • Flèches: déplacement sur la carte

Exigences système

Le navigateur cartographique du SATC est conçu pour fonctionner dans différents navigateurs web et aucun module d'extension n'est nécessaire. L'utilisation d'applications externes est nécessaire pour accéder aux données reliées au navigateur cartographique (par exemple, l'utilisateur aura besoin d'une application externe pour afficher les images des plans d'arpentages).

  • JavaScript doit être activé
